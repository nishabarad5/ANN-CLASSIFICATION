
# Customer Churn Prediction using ANN

## Overview
This project predicts customer churn using an Artificial Neural Network (ANN) built with TensorFlow/Keras and deployed using Streamlit.

## Features
- ANN-based binary classification
- Data preprocessing pipeline
- Feature scaling and encoding
- Interactive Streamlit web app
- Real-time churn prediction

## Tech Stack
- Python
- TensorFlow / Keras
- Pandas & NumPy
- Scikit-learn
- Streamlit

## Project Structure
```bash
├── experiments.ipynb
├── prediction_app.py
├── model.h5
├── scaler.pkl
├── label_encoder_gender.pkl
├── onehot_encoding_geo.pkl
```

## Run Locally
```bash
streamlit run prediction_app.py
```

## Model Workflow
1. Data Cleaning
2. Feature Encoding
3. Feature Scaling
4. ANN Model Training
5. Prediction & Deployment
